
# 软件需求规格说明书 (SRS) - 智能审查助手

## 1. 引言
### 1.1 项目背景
在企业内控和合规管理中，业务方案、合同及制度的审查是一项高频且严谨的工作。传统人工审查存在效率低、规则覆盖不全、标准不统一等痛点。本系统旨在利用大语言模型（LLM）的深度理解能力，结合企业本地规则库，构建一个集“知识管理、合规审查、内容提炼、创意可视化”于一体的智能化辅助工具。

### 1.2 文档目的
本文档详细描述了智能审查助手的各项功能及非功能需求，为后续开发、测试及验收提供基准。

## 2. 角色定义
| 角色 | 描述 | 主要职责 |
| :--- | :--- | :--- |
| **审查专家 / 管理员** | 业务规则的制定者 | 维护规则知识库、上传标准制度、启停特定规则。 |
| **业务人员 / 审查员** | 实际业务的执行者 | 上传待审材料、运行 AI 智能分析、获取提炼总结与创意演示。 |

## 3. 功能需求

### 3.1 规则知识库 (Rule Base)
*   **FR-1.1 规则录入**：支持手动录入规则标题、正文及分类。
*   **FR-1.2 文档导入规则**：支持通过上传 .pdf, .docx, .txt 文件自动提取文本并填充为新规则。
*   **FR-1.3 规则调度管理**：用户可一键开启或关闭特定规则，仅“启用”状态的规则会进入 AI 的上下文窗口。
*   **FR-1.4 智能问答中心 (Knowledge Lab)**：基于 Top-N 语义召回技术，支持用户针对已入库制度进行自然语言问答咨询。

### 3.2 审查工作台 (Review Workspace)
*   **FR-2.1 多格式材料上传**：支持对待审查的业务方案进行多文件上传（支持 PDF、DOCX、TXT）。
*   **FR-2.2 智能摘要生成**：自动提取方案核心目标、范围及主要干系人，支持用户提示词引导。
*   **FR-2.3 风险点提取 (Extraction)**：依据规则库，结构化提取关键数据点，并判定风险等级（高/中/低）。
*   **FR-2.4 审查意见草拟**：自动生成包含“审查结论、风险警示、整改建议”的正式报告。

### 3.3 内容提炼与创意可视化 (Content Distillation & Visuals)
*   **FR-3.1 多维度提炼模式**：
    *   **执行摘要**：生成高度专业、逻辑清晰的文本报告。
    *   **关键要素**：提取核心关键词、数据点和结论。
    *   **思维导图**：生成层级清晰的 Markdown 嵌套大纲。
    *   **SWOT 分析**：针对方案的优势、劣势、机会、威胁进行深度解构。
    *   **画意演示 (PPT)**：生成单页幻灯片式的视觉演示稿。
*   **FR-3.2 创意视觉生成**：
    *   **视觉隐喻**：基于提炼内容，利用 Gemini 2.5 Flash Image 模型生成与之语义关联的 3D 抽象艺术背景/插图。
    *   **信息图表化**：针对 SWOT 模式，提供 2x2 矩阵样式的可视化看板。
*   **FR-3.3 结果导出**：
    *   支持提炼结果以 Markdown 文档导出。
    *   支持创意视觉稿（海报、幻灯片插画）以 PNG 图片格式下载。

### 3.4 系统底层支撑
*   **FR-4.1 多模型引擎适配**：支持 Google Gemini 3 系列与 DeepSeek V3 引擎。
*   **FR-4.2 极速 OCR**：利用 Gemini 3 Flash 的多模态能力实现毫秒级文字提取。

## 4. 非功能需求
*   **NFR-4.1 UI 设计**：采用紧凑的图标导航栏设计，最大化内容展示区域；使用玻璃拟态 (Glassmorphism) 与 3D 渲染美学提升专业感。
*   **NFR-4.2 响应速度**：提炼任务应在 10 秒内启动，视觉生成任务应在 20 秒内完成。
*   **NFR-4.3 易用性**：提炼总结模块支持“文本模式”与“创意模式”的一键无缝切换。
